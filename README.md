# Open LLM Translator

- [x] support Google translate
- [x] support Ollama
- [x] support Deepl (need api key)
- [ ] support Cloudflare (need account id & api key)
- [ ] support OpenAI


## Technologies

- 🐍 Python 
- ⚡️ Fastapi
- 🦙 Ollama


## Build

```.shell

git clone 
cd 

pip install -r requirements/base.txt

cp env.example .env

fastapi run app/main.py

```
